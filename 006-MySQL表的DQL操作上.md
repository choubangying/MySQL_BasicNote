## MySQL的DQL操作上--单表简单查询

1. 简单查询的查询格式如下

```sql
select 列名1,列名2,列名n form 表名；

//列名可以是全部列名的一部分，使用*可以通配
select id,name,price from product;
等价于：
select * from product;
```

2. 查询结果去重

```
//某些查询结果可能有很多重复的部分，这时可以在改列名前加distinct
select price from product;
//结果如下：
+-------+
| price |
+-------+
|    20 |
|    20 |
|    30 |
|    30 |
|    30 |
|    40 |
|    40 |
|    20 |
|    50 |
|    50 |
|    50 |
|    60 |
|    60 |
+-------+
//添加去重
select distinct price from product;
//结果如下：
+-------+
| price |
+-------+
|    20 |
|    30 |
|    40 |
|    50 |
|    60 |
+-------+
```

3. 对查询结果使用数学运算

```
可以对查询结果可以使用数学操作(+ - * /)
使用方式如下：
select name,price*0.5 from product;
结果如下：
+-----------+-----------+
| name      | price*0.5 |
+-----------+-----------+
| 罗技m1    |      10.0 |
| 罗技m2    |      10.0 |
| 罗技m3    |      15.0 |
| 罗技m4    |      15.0 |
| 罗技m5    |      15.0 |
| 罗技m6    |      20.0 |
| 罗技m7    |      20.0 |
| 罗技m8    |      10.0 |
| 罗技m9    |      25.0 |
| 罗技m10   |      25.0 |
| 罗技m11   |      25.0 |
| 罗技m12   |      30.0 |
| 罗技m13   |      30.0 |
+-----------+-----------+
运算的元素可以都是表的列，可以是列和其它数值
```

4. 设置列的别名

```sql
//如下为id列设置别名pid
select id as pid,name,price from product;
其中的as可省略，若用字符串使用''包裹
```

5. 设置查询结果显示格式

```sql
使用concat函数连接字符串来设置显示格式方便客户查看数据
concat(str1,str2,...)
查询语句：
```
